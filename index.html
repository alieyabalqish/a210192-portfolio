✅ Features Implemented

📡 REST API

Live job listings from Glints API via Retrofit.

📍 GPS / Location

Find jobs within 50km of user location.

💾 Room Database

Save applied & saved jobs locally (offline).

☁️ Firebase Firestore

Cloud storage for company reviews.

📄 Resume Upload

Upload and attach resume to job applications.

💬 Chat System

Direct messaging between users and companies.

💭 Reflection: Throughout this project, I deepened my understanding of modern Android development with Jetpack Compose. I learned how to integrate multiple data sources — REST APIs, Room Database, and Firebase Firestore — into a single cohesive app. The biggest challenge was combining all four pillars: API, Sensor (GPS), Local Persistence (Room), and Cloud (Firestore). Debugging API calls and managing state with Flow and StateFlow were key learning points. I also gained experience in UI/UX design, permission handling, and navigation with Compose. This project connects to SDG 1 (No Poverty) by helping users access job opportunities, apply efficiently, and share valuable company insights with the community.
