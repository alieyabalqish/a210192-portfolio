📱 Project 2 — KerjaNow

App Name: KerjaNow

SDG: SDG 1 — No Poverty

Description: KerjaNow is a mobile job search application that helps users find job opportunities, apply with resumes, share company reviews, and connect with employers. Built with Jetpack Compose, it integrates Room Database, Firebase Firestore, and real-time APIs.

📝 Reflection & Learning Journey

Throughout this project, I deepened my understanding of modern Android development with Jetpack Compose. I learned how to integrate multiple data sources — REST APIs, Room Database, and Firebase Firestore — into a single cohesive app.

The biggest challenge was combining all four pillars: API, Sensor (GPS), Local Persistence (Room), and Cloud (Firestore). Debugging API calls and managing state with Flow and StateFlow were key learning points. I also gained experience in UI/UX design, permission handling, and navigation with Compose.

This project connects to SDG 1 (No Poverty) by helping users access job opportunities, apply efficiently, and share valuable company insights with the community.
