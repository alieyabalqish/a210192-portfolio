PROJECT 02

KerjaNow

📌 Problem Statement

Job seekers often lack access to real-time job listings, personalized job recommendations based on location, and a platform to manage applications efficiently. KerjaNow solves this by integrating live job data from external APIs, GPS-based job discovery, local job saving (Room), community reviews (Firebase), and direct chat with employers.

SDG Theme: SDG 1 — No Poverty

Kotlin Jetpack Compose Room Firebase Retrofit GPS

🏠 Home Screen

🔗 GitHub Repo → 🎥 Watch VSR →

✅ Features Implemented (4 Pillars)

  • 📡 REST API: Live job listings from Glints API via Retrofit
  • 📍 GPS / Location: Find jobs within 50km of user location
  • 💾 Room Database: Save applied & saved jobs locally (offline)
  • ☁️ Firebase Firestore: Cloud storage for company reviews
  • 📄 Resume Upload: Upload and attach resume to job applications
  • 💬 Chat System: Direct messaging between users and companies
💭 Reflection: Throughout this project, I deepened my understanding of modern Android development with Jetpack Compose. I learned how to integrate multiple data sources — REST APIs, Room Database, and Firebase Firestore — into a single cohesive app. The biggest challenge was combining all four pillars: API, Sensor (GPS), Local Persistence (Room), and Cloud (Firestore). This project connects to SDG 1 (No Poverty) by helping users access job opportunities efficiently.
