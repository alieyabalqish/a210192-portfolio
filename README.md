# A210192_AlieyaBalqish_DrNiza_Project2

## 📱 Project 2 - KerjaNow

**KerjaNow** is a mobile job search application built with Jetpack Compose for Android. It helps users find job opportunities, apply with resumes, share company reviews, and connect with employers.

---

### 🎯 SDG Theme

**SDG 1: No Poverty**

> "> "End poverty in all its forms everywhere. By providing access to employment opportunities, the app helps individuals secure stable income and break the cycle of poverty."

---

### 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| **Kotlin** | Main programming language |
| **Jetpack Compose** | UI Framework |
| **Room Database** | Local persistence (offline) |
| **Firebase Firestore** | Cloud storage for reviews |
| **Retrofit** | REST API integration |
| **GPS / Location** | Jobs near me feature |
| **Coroutines & Flow** | Asynchronous programming |
| **Material 3** | UI Design |

---

### 📱 Features Implemented

| Feature | Description |
|---------|-------------|
| ✅ **Login & Sign Up** | User authentication with email & password |
| ✅ **Job Listings** | Live jobs from Glints API |
| ✅ **Search Jobs** | Search by keyword and location |
| ✅ **Jobs Near Me** | GPS-based job search within 50km |
| ✅ **Apply Job** | Apply with resume upload |
| ✅ **Save Job** | Bookmark jobs for later |
| ✅ **Company Reviews** | Share and view company reviews (Firestore) |
| ✅ **Chat System** | Direct messaging with companies |
| ✅ **Resume Upload** | Upload and manage resumes |
| ✅ **AI Assistant** | Chat with AI for job search help |
| ✅ **Dark/Light Theme** | Theme toggle support |

### 📝 Reflection

Throughout this project, I deepened my understanding of modern Android development with Jetpack Compose. I learned how to integrate multiple data sources — REST APIs, Room Database, and Firebase Firestore — into a single cohesive app.

The biggest challenge was combining all four pillars: **API, Sensor (GPS), Local Persistence (Room), and Cloud (Firestore)**. Debugging API calls and managing state with Flow and StateFlow were key learning points.

This project connects to **SDG 8 (Decent Work and Economic Growth)** by helping users access job opportunities, apply efficiently, and share valuable company insights with the community.
